npm 包管理工具

npm安装及镜像

  • 安装node的时候会默认包含npm
    • npm install npm@latest -g 更新npm
    • npm -v 查看npm版本

  • 设置镜像

  • 或者安装cnpm镜像(淘宝镜像下载较快)：npm install -g cnpm --registry=https://registry.npm.taobao.org

    • cnpm install <module-name> 安装模块

基本命令

npm版本管理

  • package.json版本
  • 命令行修改版本号(执行命令会读取并修改package.json中的版本)

nrm 镜像管理工具

  • nrm 是一个 npm 源管理器，允许你快速地在 npm源间切换

  • 设置npm镜像为taobao镜像

  • nrm使用

mirror-config-china

  • 大部分组件通过npm设置为淘宝镜像即可加速，但是像electron-mirror、node-sass等组件需要额外设置镜像地址配置到~/.npmrc才能成功下载，此插件将常用组件的镜像地址全部加入到了上述文件夹

npx Node包执行工具

  • npm 从5.2版开始，增加了 npx 命令，用来执行Node包命令
    • npm内置此工具，或手动安装npm install -g npx
  • 使用 ^1

yarn 包管理工具

  • 官网：https://yarnpkg.com/zh-Hans/
  • 安装 npm install -g yarn，通过官网的msi容易报’yarn’ 不是内部或外部命令，也不是可运行的程序
  • 类似于npm，基于package.json进行包管理

  • 设置镜像

  • 使用

nvm Node版本管理工具

  • nvm全名node.js version management，顾名思义是一个nodejs的版本管理工具，通过它可以安装和切换不同版本的nodejs
  • 下载安装
    • windows下载，安装之前可能需要先卸载之前安装的Node
    • Unix: curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.38.0/install.sh | bash (Mac M1 11.4安装成功，但是安装node v10.x失败，v12.x成功)
  • 使用

vue 命令行工具(vue-cli)

  • 参考vue.md#vue-cli
